# How to contribute to the dictionary data

Please edit the dictionary and/or the lemma mapping file:

* [docs/dic.ja.json](docs/dic.ja.json)
* [docs/word2lemma.json](docs/word2lemma.json)

# How to build

```
$ make
$ ./preview
```
